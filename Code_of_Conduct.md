# Code of Conduct - AIC-SSI

We expect all contributors to act with respect, honesty, and integrity.

**Key Rules:**
- No harassment, discrimination, or personal attacks.
- Respect self-sovereignty and user privacy.
- Do not propose features that compromise ethical principles or third path independence.
- Any violation may result in removal from the project.

This project follows the Contributor Covenant with the addition of AIC’s core principles (zero-donation, third path absolute, ethical-from-kernel).
